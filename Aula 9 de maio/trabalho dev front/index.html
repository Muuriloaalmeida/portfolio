Introdução ao Desenvolvimento Front-end

O desenvolvimento front-end é responsável por criar a interface gráfica com a qual os usuários interagem em um site ou aplicação web. Envolve o uso de tecnologias como HTML, CSS e JavaScript para estruturar, estilizar e dar interatividade às páginas. Um bom front-end proporciona uma experiência visual atrativa e funcional.

×

Tabelas

As tabelas em HTML são utilizadas para organizar e exibir dados em formato tabular, com linhas e colunas. A estrutura básica envolve as tags <table>, <tr>, <th> e <td>. Apesar de hoje serem mais utilizadas para dados, no passado as tabelas também eram usadas para layout, prática atualmente desencorajada.

×

Formulários

Formulários são elementos fundamentais para a coleta de dados dos usuários. Utilizando tags como <form>, <input>, <select> e <textarea>, os desenvolvedores criam interfaces que permitem ações como login, cadastro, envio de mensagens e mais. A validação dos campos é essencial para garantir a integridade das informações.

×

Estilos de CSS

O CSS (Cascading Style Sheets) é a linguagem de estilos usada para definir cores, fontes, espaçamentos, alinhamentos e muito mais nos elementos HTML. Ele separa a estrutura (HTML) da apresentação visual, permitindo criar interfaces bonitas, consistentes e responsivas com eficiência.

×

Media Query

As media queries são recursos do CSS que permitem aplicar estilos diferentes dependendo do tamanho da tela ou dispositivo. São essenciais para o design responsivo, garantindo que o conteúdo se adapte bem a celulares, tablets e desktops. Exemplos comuns incluem mudanças em layout, tamanhos de fonte e exibição de elementos.

Esse projeto
tem como objetivo
mostrar o conteúdo das aulas

Este projeto reúne os principais tópicos abordados durante as aulas de introdução ao desenvolvimento front-end. O conteúdo foi apresentado em sala com o objetivo de proporcionar uma compreensão inicial sobre a construção de interfaces web modernas. As aulas exploram a criação de pás utilizando tabelas, formulários, estilos com CSS e técnicas de responsividade com media queries. A navegação permite revisar cada tema de forma organizada e objetiva.
